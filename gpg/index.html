How to Send Me an Encrypted Message

0. Install gnupg if you don't already have it

1. Import my public key

gpg --import

# Press CTRL+d after pasting it into terminal

2. Verify the fingerprint

gpg --fingerprint roni.kobrosly@proton.me

3. Encrypt it

gpg --encrypt --sign --armor -r roni.kobrosly@proton.me

# Press CTRL+d when finished
